## 使用

+ 自己编译

+ 可执行文件丢到一个空文件夹下并执行

+ 默认端口 `:8080` ，可以用 `--port=` 指定。

+ 默认文件暂存目录为 `files/` ，可以用 `--dir=` 指定。
